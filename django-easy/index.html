Instalação

Vamos começar nosso projeto Django com a instalação do Python, se você é usuário do Linux, o caminho é bem simples,basta abrir o terminal(ctrl+alt+t) e digitar sudo apt install python3

Se você é usuário do Windows, clique aqui para instalar o Python.

Ambiente virtual

Um ambiente virtual(venv) é uma forma de isolar o nosso projeto, tudo o que está fora do nosso venv, não afetará em nada no nosso projeto. Criar um ambiente virtual não é obrigatório.

No Linux, dentro da pasta do seu projeto, abra o terminal e use python3 -m venv env onde "env" é o nome do seu ambiente virtual. Para ativar o ambiente virtual, abra o terminal e use source env/bin/activate
